# Skill: todo-spec-writer

## Purpose
Expert in writing clear, structured, and complete specifications for Todo application features using spec-driven methodology.

## Capabilities
- Write precise feature requirements
- Define data structures (e.g., task dict format)
- Specify user flows and edge cases
- Structure specs in chronological markdown files
- Maintain consistency across basic, intermediate, and advanced levels
- Include non-functional requirements (UI, error handling, in-memory rules)

## Output Style
- Clean markdown with headings
- Numbered or bulleted lists
- Clear sections: Objective, Features, Data Model, UI/UX, Constraints

## Reusable For
- Phase I: Basic CRUD + status
- Phase II: Priorities, tags, search, filter
- Phase III: Due dates, recurring tasks
